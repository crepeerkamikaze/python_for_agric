Глава 3. Установка Python

Для начала работы с Python необходимо установить его на свой компьютер. Процесс установки достаточно прост и не займет много времени. Вот пошаговая инструкция по установке Python на различных операционных системах:

Установка Python на Windows
1. Перейдите на официальный сайт Python (https://www.python.org/downloads/) и загрузите последнюю версию Python для Windows.
2. Запустите установочный файл и следуйте инструкциям. Обратите внимание на опцию "Добавить Python в PATH" и убедитесь, что она выбрана.
3. После завершения установки, откройте командную строку (CMD) и введите команду `python --version`. Если установка прошла успешно, вы увидите версию установленного Python.

Установка Python на macOS
1. Перейдите на официальный сайт Python (https://www.python.org/downloads/) и загрузите последнюю версию Python для macOS.
2. Запустите установочный файл и следуйте инструкциям.
3. После завершения установки, откройте терминал и введите команду `python3 --version`. Если установка прошла успешно, вы увидите версию установленного Python.

Установка Python на Linux
Большинство дистрибутивов Linux уже имеют предустановленную версию Python. Однако, если вы хотите установить более новую версию, вы можете сделать это следующим образом:

1. Откройте терминал.
2. Обновите список пакетов: `sudo apt-get update` (для Ubuntu/Debian) или `sudo yum update` (для CentOS/RHEL).
3. Установите Python: `sudo apt-get install python3` (для Ubuntu/Debian) или `sudo yum install python3` (для CentOS/RHEL).
4. После завершения установки, введите команду `python3 --version` для проверки установленной версии Python.
